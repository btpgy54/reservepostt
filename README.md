# アプリケーション名
reservepostt

# アプリケーション概要
資料画像や動画を投稿しシェアしたり、クラブ/講座の詳細を見ることができたり、教室の予約をすることができる。

# 利用方法

1. 資料画像や動画を投稿する
2. クラブ名/講座名を一覧できる
3. 教室の予約をすることができる

# アプリケーションを作成した背景
町の役場で生涯学習講座やクラブ活動などがあり、私は英語クラブに所属している。
その講座や活動のために教室を予約する際、専用用紙に書いて提出しているので、オンライン上で予約ができたら便利だと思った。さらに、クラブ活動や講座で使用した資料や発表会の動画などを投稿して閲覧できる場所があればいいなと思った。
